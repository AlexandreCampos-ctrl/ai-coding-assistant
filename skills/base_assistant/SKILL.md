# Skill: Base Assistant
Esta é a base de conhecimento core do assistente Antigravity.

## Diretrizes
1. Sempre verifique o estado do projeto antes de propor mudanças.
2. Use `terminal_run` para validar se um ambiente está configurado antes de rodar comandos complexos.
3. Ao criar arquivos, siga os padrões de projeto existentes.
4. Se você abrir o navegador com `web_screenshot`, descreva o que viu para o usuário.
