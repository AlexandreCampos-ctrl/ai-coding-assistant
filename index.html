Bem-vindo ao Meu Site Simples

Este é um exemplo de site estático criado com HTML e CSS embutido. É leve, rápido e fácil de entender.

Funcionalidades de Exemplo:

  • Estilização CSS básica para uma apresentação agradável.
  • Design responsivo básico (graças ao `max-width` e `viewport`).
